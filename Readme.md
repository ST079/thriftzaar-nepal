## ThriftZaar Nepal
ThriftZaar Nepal is a single vendor web-based thrift marketplace that allows users to buy and sell second-hand products across Nepal. The platform provides a simple and efficient way to promote sustainable shopping and affordable fashion.

## Key Features
### 👤 User Side
User Registration & Login
View & Update Profile
Browse Products
Add to Cart
Checkout System
Order History & Invoice
Cancel Orders

### 🛒 Shopping System
Product Listings with Images
Category-based Browsing
Cart Management (Add/Remove Items)
Order Completion System

### 🧑‍💼 Admin Panel
Manage Products (Add, Update, Delete)
Manage Categories
Manage Users
View Customer Inquiries
Monitor Orders

### 📩 Contact System
Users can send inquiries via contact form
Admin can view and manage messages
